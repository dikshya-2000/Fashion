information, nor can it guarantee the confidentiality of the same.

and services provided will fully satisfy their needs.

The Website uses data security technologies such as firewalls, access control procedures and cryptographic

Furthermore, all transactions via the Website will be performed via secure payment systems. Confidential

Berry declares that it has adopted all the technical and organisational measures necessary to guarantee the
